
# Gestion de Bebidas
Proyecto hecho con NestJS
Patrones de diseño utilizados: mvc, services , DTO, repository
BD: Postgres , ORM : Typeorm, 
se implementa swagger para documentar las apis.
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Config Global

```bash
#copiar el archivo de configuraciones globales de ejemplo a su entorno local
$ cp .env.example .env
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

documentacion de apis
http://localhost:3000/api/ 
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
